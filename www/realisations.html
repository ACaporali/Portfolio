Développement d’un annuaire web et d’une solution de localisation

Lors de ma deuxième année dans le BTS SIO, j’ai effectué un stage de six semaines à la Caisse d’Epargne de Grenoble, avec l’un de mes collègues de classe. Nous avons été chargés de la réalisation du projet "Find me INCITY". Il s'agit d’un annuaire de renseignement avec la possibilité de localiser les éléments recherchés.
Cet annuaire a pour but de renseigner rapidement les utilisateurs (collaborateurs externes/internes, et les visiteurs) qui cherchent un service afin d’éviter une perte de temps.

L'annuaire devait être présenté sous format web et en responsive design, c’est à dire qu’il s’adapte à tous types de formats (ordinateur fixe, tablette, smartphone). De plus, la localisation de tous les éléments disponibles de la tour devait se faire via une partie indépendante de l'annuaire.
Cette application était destiné à être mis en place dans les nouveaux locaux de la Caisse d'Epargne de Lyon (tour INCITY).
Pour réaliser ce projet nous avons utilisé l’éditeur de texte Notepad++ ainsi qu’une base de donné sous MySQL. Nous nous sommes essentiellement servis du HTML 5, PHP, SQL et JavaScript. Le design ainsi que le responsive design est géré par une librairie Bootstrap auquel nous avons apporté des modifications.

Développement d’une interface graphique client en C#

Dans le cadre de ma formation en BTS SIO, j’ai réalisé un stage d’un mois dans l’entreprise Axe Informatique avec l’un de mes collègues de classe. L’activité de l’entreprise est essentiellement basée sur la vente et formation sur logiciels de gestion SAGE, mais aussi sur le conseil en informatique.
Axe Informatique réalise sur demande des extensions de logiciels SAGE. Ces demandes sont propres à chacun de leurs clients. Auparavant, une fois les extensions développées, Axe Informatique englobait leurs produits développés dans une interface simple et personnalisée pour faciliter l’accès aux clients. Cependant, l’entreprise a décidé de changer l’interface et de la rendre unique à tous leurs clients, mais aussi qu’elle soit moderne et design. Ceci était un point important, puisque cela permettait à l’entreprise de bénéficier d’un avantage esthétique sur leurs concurrents.
Nous avons travaillé sur tous les modules sauf celui portant le nom : « Traitement ». Ce module est réservé aux extensions de logiciels SAGE, par conséquent il ne nous concernait pas.
Pour réaliser ce projet, nous avons utilisé le logiciel Microsoft Visual Studio Express 2013, avec le WPF (Windows Presentatation Foundation).

Projet Numérologie

Dans le cadre d’un projet personnel encadré mon groupe (constitué de quatre personnes au total) et moi avons retravaillé une application de voyance. Plus précisément, nous avons fait une migration de Java vers C#, ainsi qu’une mise à jour avec ajout de fonctionnalité.
Cette application de numérologie a pour but d’afficher une prédiction pour la voyante Mme IRMA en fonction de données qu’elle aura préalablement saisies. Les prédictions sont enregistrées dans le « Grand livre ».
Pour ce projet, nous avons utilisé le logiciel Microsoft Visual Studio 2010 avec le C# pour le langage de programmation, ainsi que les Windows Forms (WinForms) pour la parite graphique.
